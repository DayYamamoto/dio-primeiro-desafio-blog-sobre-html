Surgimento da Internet

A sigla HTML significa HyperText Markup Language e é considerada como uma linguaem padrão que é utilizada para estruturar e criar os conteúdos de uma página web.

Utiliza-se de marcadores denominado "tags" para inidicar tanto como ele vai ser exibido em um navegador quanto para determinar o conteúdo que irá aparecer

Por meio do HTML é possível definir qual será a estrutura básica da página web, acrescentando por exemplo o cabeçalho, parágrafos, tabelas, links, imagens, listas e vai servir posteriormente para estilização por meio do CSS

Quando se cria um HTML os navegadores conseguem interpretar aquele código e exibir o conteúdo de acordo com o que foi descrito no HTML.

Como funciona o HTML

O HTML funciona partir de "tags" e marcadores para marcar o conteúdo da página e definir qual será sua estrutura

O HTML em sua estrutura básica vai consistir em head, que contém informações que não aparecem para o usuário e body, onde vai ser escrito basicamente a parte do código que aparecerá para o usuário na página web

O HTML é um código base que vai criar apenas páginas web simples e de baixa complexidade e por isso é frequentemente utilizada com outras tecnologias, como o CSS para que seja estilizada e se mostre mais complexa ao usuário

O que são Tags

As "tags" sáo contidas dentro de dois sinais, um menor e um menor e são representadaspor palavras chaves que determinarão o que aquela tag representa ou faz dentro do código

As tags podem conter atributos que fornecerão informações adicionais sobre um determinado elemento, como por exemplo ao se utilizar a tag img tem-se a opção de colocar um atributo denominado src que vai especificar qual a fonte ou local da imagem que vai ser utilizada

Importante lembrar que a maioria das "tags" possuem uma tag de abertura e uma tag de fechamento, representada por /, para que funcionem. Caso a tag de fechamento seja esquecida o código não funciona da maneira correta, podendo até mesmo "quebrá-lo"
